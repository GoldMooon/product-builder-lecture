# Lotto Number Recommendation Site Blueprint

## Overview

This application is a simple, user-friendly Lotto Number Recommendation Site. It provides users with a set of randomly generated lottery numbers, offering a fun and interactive way to choose their next ticket.

## Project Outline

### Style and Design

*   **Layout:** A clean, centered layout that is mobile-responsive.
*   **Color Palette:** A vibrant and energetic color scheme to create a positive user experience.
*   **Typography:** Expressive and clear typography to guide the user.
*   **Iconography:** Use of icons to enhance visual communication.
*   **Interactivity:** Interactive elements with visual feedback (e.g., button glow effects).

### Features

*   **Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Display:** Displays the generated numbers in a clear and visually appealing format.
